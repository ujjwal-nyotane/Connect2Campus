# Connect2Campus

Connect2Campus is a static, multi-page student portal prototype built with HTML and CSS.

## Pages

- `index.html` – login and entry point
- `home.html` – dashboard/home screen
- `attendance.html` – attendance view
- `grades.html` – grades page
- `timetable.html` – timetable/schedule
- `leave.html` – leave request page
- `hostel.html` – hostel information
- `fees.html` – fees and payments
- `notifications.html` – notifications page
- `support.html` – support/help page
- `profile.html` – student profile page

## Tech Stack

- HTML5
- CSS3

## Run Locally

1. Clone this repository.
2. Open `index.html` directly in your browser.

No build step or package installation is required.
